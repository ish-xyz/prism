# PRISM - Procedural Reasoning Instruction Schema for Models

A better way to exchange information between models and humans. Still leveraging the archaic method (keyboard and mouse) but trying to be more efficient and accurate with it. 

## How to use:

re-write skills in PRISM via LLMS with the following prompt: "re-write skill xyz in PRISM using the following spec https://raw.githubusercontent.com/ish-xyz/prism/refs/heads/main/PRISM_SPEC.md"

Enjoy.
